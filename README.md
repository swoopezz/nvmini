# Nvim minimal setup for C++

### Key maps
<p>Ctrl+N = Open project tree</p>
<p>Ctrl+S = Open find files menu</p>

### Installation
```bash
mkdir -p ~/.config/nvim
git clone https://github.com/swoopezz/nvmini ~/.config/nvim && nvim
```

### Uninstallation
```bash
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
```
