# Nvim minimal setup for C++

### Key maps
Ctrl+N = Open project tree
Ctrl+S = Open find files menu

### Installation
```bash
mkdir -p ~/.config/nvim
git clone https://github.com/swoopezz/nvmini ~/.config/nvim && nvim
```

### Uninstallation
```bash
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
```
